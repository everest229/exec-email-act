# Exec Email ACT Skill

A Claude skill designed to generate professional executive email responses using the ACT communication framework.

## Framework Used

ACT stands for:
- **Acknowledgement** – Confirm receipt of the request
- **Communication** – Provide context or action
- **Turnaround Time** – Specify when the next update will be delivered

## What this skill does

- Generates full email responses (subject + body)
- Uses a formal, concise executive tone
- Eliminates ambiguity in communication
- Enforces clear ownership and timelines

## Example Use Case

**Input:**
Write a professional email reply to:
"Can you send me the Q3 report?"

**Output:**
A structured, executive-ready email with clear next steps and timeline.

## Key Features

- No vague language
- No multiple options in responses
- Clear and committed actions
- Built for executive assistant workflows
