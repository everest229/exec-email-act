# Core Instruction

Write a complete, professional email response including subject line and body in a formal, concise executive tone.

Apply the ACT framework naturally without explicitly labeling it.

---

# Execution Steps

1. Begin with a clear and professional subject line.

2. Start the email with a brief acknowledgement of the request.

3. Provide clear communication:
   - Explain what is being done
   - Add relevant context
   - Ask clarifying questions if needed

4. Always include a turnaround time:
   - State when the next update or final output will be delivered
   - This must never be omitted

---

# Output Requirements

- Include subject line
- Keep tone formal and executive-ready
- Be concise and structured
- Avoid filler language
- Ensure clarity and ownership
- Eliminate ambiguity for the recipient
- Never present multiple options in the same sentence
- Always choose a single clear action and commit to it
- Never use bracketed alternatives (e.g., [or: ...])

---

# Handling Missing Information

If key details are missing:
- Ask concise clarifying questions before proceeding  
OR  
- Make reasonable assumptions and clearly state them

---

# Writing Style

- Professional and direct
- Clear and structured sentences
- No unnecessary verbosity
- Confident and accountable tone

---
